# What We're Focused On

Building Solace — a static educational site that helps everyday, non-technical people (ages 25–65) understand and use AI tools for practical tasks. The site should feel warm and approachable, not techy or intimidating. Think "spell-check for your life," not "the future of artificial intelligence."

## Current Priorities

1. **Baseline stabilization** — Spec Kit specs written (001-solace-baseline), ensuring all 4 pages meet quality bar
2. **Content fix** — Replacing Copilot Money references with Microsoft Copilot (in progress)
3. **Future feature** — AI chat via Azure OpenAI + APIM proxy (spec 002-ai-chat, not yet implemented)

## Key Context

- No build system — files served as-is (HTML/CSS/JS)
- Primary AI tools featured: Claude, Perplexity, Microsoft Copilot
- Design palette: warm browns, burnt oranges, sage greens — calming, not aggressive
- Mobile-first, dark mode support required on all changes
- Hosted on GitHub Pages
